①描述性分析旨在快速回答“数据长什么样”：有多少条记录？平均水平如何？波动大不大？极值是多少？常用统计指标包括 count、mean、std、min、max 等；在 pandas 中可通过 df.describe() 一次性查看，或使用 agg() 按需组合多个统计量。

确定分组依据（定性如“年份”“地区”，定量如“票价区间”）；

使用 df.groupby("分组字段") 进行分组；

对分组后的对象调用聚合函数（如 mean()、sum()、count()）完成统计。

某超市推出苹果促销活动，每斤苹果定价 12.8 元。购买规则根据重量分段设置不同折扣：

购买重量小于等于 1 斤：无折扣，按原价结算；

购买重量大于 1 斤且小于等于 5 斤：享受 9 折优惠；

购买重量大于 5 斤且小于等于 10 斤：享受 8 折优惠；

购买重量大于 10 斤：享受 7 折优惠。

用户通过键盘输入购买数量，屏幕上输出原价和折后价（保留两位小数）。

写出 python 代码

 

# 获取用户输入的重量（浮点数）

weight = float(input("请输入购买苹果的重量（斤）："))

 

# 单价定义

price_per_kg = 12.8

 

# 根据重量区间应用不同折扣率

if weight <= 1:

    discount_rate = 1.0  # 无折扣

elif weight <= 5:

    discount_rate = 0.9  # 9折

elif weight <= 10:

    discount_rate = 0.8  # 8折

else:

    discount_rate = 0.7  # 7折

 

计算原价和折后价

original_price = weight * price_per_kg

final_price = original_price * discount_rate

输出结果（保留两位小数）

print (f"原价：{original_price:.2f} 元")

print (f"折后价：{final_price:.2f} 元")
